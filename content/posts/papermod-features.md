## PaperMod Theme

PaperMod is a fast, clean, and responsive Hugo theme.

### Features

*   Dark Mode
*   Fast Search
*   Multilingual Support
*   Archives Page
*   Tags Page

Check out the [documentation](https://github.com/adityatelange/hugo-PaperMod/wiki/Features) for more info.
